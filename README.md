# 🏋️ 내 손안의 PT 선생님
> AI 기반 홈 트레이닝 자세 교정 및 운동 카운터

---

## 📌 프로젝트 소개

웹캠 영상을 입력받아 사용자의 주요 신체 관절(Keypoints)을 실시간으로 추적하고,  
스쿼트·푸시업 등의 운동 자세가 정확한지 분석하는 AI 기반 홈 트레이닝 보조 프로그램입니다.

- ❌ 잘못된 자세 감지 시 → 화면에 경고 메시지 표시
- ✅ 올바른 자세일 때만 → 운동 횟수 자동 카운트
- 💪 별도 장비 없이 웹캠 하나로 나만의 PT 선생님!

---

## 🎯 주요 기능

| 기능 | 설명 |
|------|------|
| 실시간 자세 추적 | MediaPipe 또는 YOLOv8-Pose를 활용한 스켈레톤(관절) 검출 |
| 자세 정확도 분석 | 관절 각도 계산을 통한 올바른 자세 판별 |
| 자동 운동 카운터 | 올바른 자세일 때만 횟수를 카운트 |
| 시각적 피드백 | 화면에 스켈레톤 오버레이 및 경고/승인 메시지 표시 |

---

## 🛠️ 기술 스택

- **언어**: Python
- **핵심 라이브러리**:
  - [MediaPipe](https://mediapipe.dev/) / [YOLOv8-Pose](https://docs.ultralytics.com/) — 포즈 추정
  - OpenCV — 웹캠 영상 처리 및 시각화
  - NumPy — 관절 각도 계산
- **실행 환경**: 웹캠이 연결된 로컬 PC
